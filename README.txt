# Raport Laravel System
A backend-heavy reporting system built with Laravel 10+, Tailwind CSS, and MySQL.

### Features
* **Full CRUD**: Manage subjects (Mapel) with specific categories (Wajib, Umum, Produktif).
* **Database Logic**: Relational data management for student grades.
* **Modern Stack**: Developed using Laravel Breeze & Blade components.

### Tech Stack
* **Backend**: PHP 8+, Laravel
* **Frontend**: Tailwind CSS
* **Database**: MySQL (XAMPP)
